# Strategic Planner

A focused plugin for strategic planning consultants. Runs complete engagements from scoping through research, rigorous analysis, synthesis, and executive-grade deliverables.

## What It Does

1. **Scope** the strategic challenge with the client
2. **Research** with two independent agents + validation
3. **Analyze** with built-in decision science (MAP scoring, pre-mortem, reality check)
4. **Synthesize** a coherent storyline
5. **Deliver** an executive-grade .docx report

## Commands

| Command | Purpose |
|---------|---------|
| `/plan <brief>` | Run a complete strategic planning engagement |
| `/research <topic>` | Standalone research (two agents + validation) |
| `/review` | Review past predictions and calibration accuracy |

## Skills

| Skill | Purpose |
|-------|---------|
| **scoping** | Define and sharpen the strategic challenge |
| **research** | Parallel research with validation |
| **analysis** | Decision science engine (MAP, pre-mortem, reality check) |
| **synthesis** | Build the strategic storyline |
| **deliver** | Generate the executive-grade .docx report |
| **docx** | Word document formatting engine |

## Agents

| Agent | Role |
|-------|------|
| **researcher-a** | Direct research angle (market data, financials, benchmarks) |
| **researcher-b** | Complementary angle (competitive dynamics, case studies, contrarian) |
| **validator** | Cross-checks both researchers for consistency and source quality |
| **challenger** | Decision science engine — MAP scoring, pre-mortem, reality check |
| **calibrator** | Tracks predictions and measures calibration over time |

## Design Principles

- **Decision science built in** — not an add-on, runs at every analytical step
- **Evidence-traced** — every claim traces back to a sourced finding
- **Bias-resistant** — two independent research threads + validation + challenge
- **No clichés** — clear, direct executive writing
- **Expert-in-the-loop** — 80-90% accelerator, not a replacement for judgment
- **Honest uncertainty** — confidence scores and base rates always surface

## Setup

1. Install the plugin in Claude Desktop or Cowork
2. Run `/plan` with your strategic challenge
3. The plugin guides you through the 5-phase workflow
