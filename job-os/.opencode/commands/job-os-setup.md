# /job-os-setup

Run the job-os onboarding wizard.

Loads `modes/setup.md` and guides you through:
- Selecting your target tracks (AI Engineer, UX Designer, etc.)
- Creating a CV for each track
- Configuring your profile and narrative
- Setting up the portal scanner
- Validating everything with a health check

Safe to re-run at any time — existing applications, reports, and pipeline data are never touched.

---

Invoke the job-os skill with the `setup` sub-command:

Read the file `modes/setup.md` and follow it exactly, starting at Step 1.
