# AntiSlop Lexicon EN — Corporate Fluff & AI Markers

## Corporate Slop / Hollow Jargon

### Level 1 — High Severity (Pure Slop)

- in today's fast-paced world
- in the ever-evolving landscape
- it is crucial to / it is essential to / it is paramount to
- it is important to note that
- it is worth noting that
- game-changer / game-changing
- cutting-edge / state-of-the-art / next-gen
- synergy / synergistic
- holistic approach / holistic view
- paradigm shift
- thought leader / thought leadership
- disruptive / disruption (when not about actual tech)
- ecosystem (when not biology)
- scalable / scalability (without metrics)
- digital transformation
- value proposition
- best-in-class / world-class
- moving the needle
- low-hanging fruit
- circle back / close the loop
- take it to the next level
- deep dive
- at the end of the day
- unlock value / unlock potential
- stakeholder alignment
- core competency
- mission-critical

### Level 2 — Moderate Severity (Common Corporate, Human use possible)

- optimize / optimization (without specifying what/how)
- leverage (vague use)
- impactful
- empower / empowerment
- streamline / streamlined
- innovative / innovation (without description)
- robust / robust solution
- seamless / seamlessly
- actionable / actionable insights
- drive growth / drive results
- foster / encourage (vague)
- navigate (metaphorical)
- resilient / resilience
- sustainable / sustainability (greenwashing context)
- inclusive / inclusivity (without action)
- strategic (as empty intensifier)
- roadmap (without substance)
- engagement (undefined)

### Level 3 — Low Severity (Suspect in clusters)

- implement / implementation
- utilize (instead of "use")
- facilitate
- strive to
- ensure that
- enhance / enhancement
- comprehensive / exhaustive
- in terms of
- regarding / with regard to
- going forward
- at this stage / juncture
- due to the fact that (instead of "because")

## Filler Connectors (LLM Padding Signals)

### Delete in 90% of cases

- furthermore
- moreover
- in addition
- additionally
- it goes without saying that
- needless to say
- it is interesting to note that
- in fact / as a matter of fact
- similarly / likewise
- having said that / that being said

### Suspect in clusters

- indeed
- thus
- therefore
- however (when contrast is weak)
- nevertheless (when no real opposition follows)

## Vague Verbs (Replace with Specific Action)

| Vague Verb | Typical Replacement                            |
| :--------- | :--------------------------------------------- |
| optimize   | reduce processing time by 30%, automate step Y |
| transform  | replace A with B, migrate from X to Y          |
| improve    | measure Z and hit threshold T                  |
| strengthen | add control X, train N people                  |
| leverage   | use tool X to achieve Y                        |
| streamline | cut steps from 7 to 3, remove approval Z       |
| innovate   | introduce technique X not used before          |
| empower    | train, fund, give tool X to                    |
| drive      | increase metric X from A to B by doing Y       |

## AI Writing Tics (Claude/GPT Markers)

### Hedging Clusters

- it is important to consider
- while there are many factors
- it is a complex issue
- there are multiple perspectives
- it depends on various factors
- opinions may vary

### False Balance

- on the one hand... on the other hand (when one side has zero evidence)
- there are pros and cons to each approach
- reasonable people may disagree

### Over-Summarization

- in summary / to summarize / in short (when no summary is needed)
- the key takeaway is
- the bottom line is

### Obsequious Openers

- that is an excellent question!
- you raise a very interesting point
- absolutely!
- certainly!
