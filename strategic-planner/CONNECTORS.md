# Connectors

This plugin works without any external integrations.

## Microsoft 365 (optional)

- **OneDrive** — Store and retrieve client briefs and deliverables
- **Word / Excel** — Direct document creation

Connect via Settings → Connectors if not active.

## Usage

When the deliverable skill produces a .docx, it saves locally. If a cloud connector is active, it can also save to OneDrive.
