# Préparation d'Entretien de Découverte

Cette compétence vous aide à concevoir des plans d'entretien efficaces pour maximiser l'apprentissage et éviter les biais.

## 🛠 Usage

À utiliser AVANT de parler à un client pour structurer votre approche (Mom Test, JTBD, Switch).

## 📂 Ressources (Progressive Disclosure)

- [Guide de Préparation Complet](./references/full-guide.md) : Méthodologie, types d'entretiens, biais et critères de réussite.

## 🚀 Workflow

1. Définir l'objectif de recherche (Validation, JTBD, Churn).
2. Identifier le segment cible.
3. Choisir la méthodologie (Mom Test, Switch, Journey Map).
4. Générer le guide d'entretien sur mesure.
